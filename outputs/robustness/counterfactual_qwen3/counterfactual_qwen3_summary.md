# Qwen3-8B QLoRA 反事实鲁棒性实测

该实验使用 FINNSP 上微调得到的 Qwen3-8B QLoRA 主模型，对自动构造的反事实样本进行真实推理。四类扰动分别考察主体替换、负面触发词删除、干扰主体插入和否定/澄清语境。该结果用于压力测试和边界分析，不作为主任务核心性能指标。

| 指标 | 结果 |
|---|---:|
| entity_switch_accuracy | 0.4500 |
| negative_cue_sensitivity | 0.0400 |
| distractor_resistance | 0.9500 |
| negation_robustness | 0.9400 |
| invalid_json_rate | 0.0000 |
| avg_latency_sec | 1.3550 |

总样本数：400。模型对干扰主体和否定/澄清语境较稳，但主体替换和负面触发词删除仍是明显短板。原因包括：自动替换主体后，文本中可能仍保留其他真实风险主体；仅删除少量显式负面词后，文本中仍可能保留风险提示、备案异常等弱风险线索。因此，该实验应作为压力测试而非人工标注鲁棒性基准。
